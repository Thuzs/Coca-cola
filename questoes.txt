3. O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
O git add prepara os aquivos selecionados para serem afiliados a um commit, a diferença é que apenas o add da para selecionar qual o arquivos em expecifico que você quer adicionar, e o "add ." adiciona todos

4. Por que utilizamos o ponto (.) no comando git add .?
Pois é mais rápido para não ter que esrever arquivo por arquivo
